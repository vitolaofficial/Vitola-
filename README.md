# Vitola App - The Sommelier Guided Cigar Experience

Experience the art of the leaf with Vitola, a premium digital companion designed for cigar aficionados.

## Key Features

- **Pairing Sommelier**: Interactive drink-to-cigar and cigar-to-drink recommendations with visionary flavor architecture visualization.
- **Digital Humidor**: Manage your collection with detailed metadata, aging tracking, and tasting notes.
- **Cigar Discover**: Search and filter through the world's finest cigars.
- **Tasting Journal**: Record your experiences with a sophisticated logging system.
- **AI Concierge**: A personal cigar expert available 24/7.

## Technologies

- **Vite** & **TypeScript**
- **React** with **Framer Motion**
- **Supabase** for Backend
- **Tailwind CSS** & **shadcn-ui**

## Getting Started

1. Clone the repository.
2. Install dependencies with `npm install`.
3. Set up your `.env` file with Supabase and RapidAPI credentials.
4. Run the development server with `npm run dev`.
